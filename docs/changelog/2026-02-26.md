## [15:30] - Phase 1: AI Service Core (Training Pipeline + Real Inference)

- **Loại**: feat

- **Files thay đổi**:
  - `scripts/fetch_data.py` — [NEW] Fetch VIC data via vnstock, save raw/processed CSV
  - `ai-service/app/domain/forecasting/probabilistic_head.py` — [NEW] Gaussian NLL loss function
  - `ai-service/app/domain/forecasting/data_scaler.py` — [NEW] MinMax scaler with JSON save/load
  - `ai-service/app/domain/forecasting/dataset.py` — [NEW] PyTorch Dataset for training samples
  - `ai-service/app/infrastructure/model_registry/loader.py` — [NEW] Model save (with archive) + load
  - `ai-service/scripts/train.py` — [NEW] Full training pipeline (NLL loss, calibration eval, metrics)
  - `ai-service/app/services/forecast_service.py` — [MODIFIED] Real data via vnstock, scaler support

- **Hàm mới tạo**:
  - `fetch_vic_data(start_date, end_date)` trong `fetch_data.py` — Lấy dữ liệu VIC qua vnstock
  - `process_features(df)` trong `fetch_data.py` — Tính indicators và lưu processed CSV
  - `GaussianNLLLoss.forward(mu, sigma, target)` trong `probabilistic_head.py` — NLL = 0.5[log(σ²) + (y-μ)²/σ²]
  - `FeatureScaler.fit_transform(df, cols)` trong `data_scaler.py` — MinMax scaling + save to JSON
  - `FeatureScaler.save(path)` / `FeatureScaler.load(path)` — Persist/restore scaler params
  - `VICStockDataset.__getitem__(idx)` trong `dataset.py` — Returns (sequence, horizon, target)
  - `save_model(model, path, archive)` trong `loader.py` — Save model + auto-archive old
  - `load_model(model_class, path)` trong `loader.py` — Load model weights
  - `train(data_path, epochs, ...)` trong `train.py` — Full training pipeline with evaluation
  - `ForecastService._get_latest_sequence()` trong `forecast_service.py` — Real VIC data for inference

- **Ghi chú**: Training pipeline hoàn chỉnh. Cần chạy `scripts/fetch_data.py` để lấy data, sau đó chạy `ai-service/scripts/train.py` để train model. Sau khi train, forecast_service sẽ tự động load model và dùng dữ liệu VIC thực.
