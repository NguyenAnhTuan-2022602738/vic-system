Chương này trình bày quá trình phân tích toán học và minh họa tính toán thủ công cho ba mô hình dự báo: (1) Hồi quy tuyến tính, (2) Rừng ngẫu nhiên, và (3) Hybrid CNN-LSTM-Attention. 3.2.2. Biến mục tiêu: Lợi nhuận phần trăm (target_return)

Không giống hệ thống cổ điển dự báo giá tuyệt đối, hệ thống VIC dự báo lợi nhuận phần trăm tại T+2 (sau 2 phiên giao dịch) — phù hợp với bài toán đầu tư thực tế, cho phép nhà đầu tư có đủ thời gian ra quyết định:

target_returnt = (Closet+2 − Closet) / Closet

Ví dụ: Nếu ngày T VIC đóng cửa ở 68.00 và hai phiên sau (T+2) là 68.90 thì target_return = (68.90 − 68.00)/68.00 = +1.32%. Đây là tín hiệu để nhà đầu tư quyết định MUA/BÁN ngay hôm nay, thực hiện ở phiên T+1, và thoát lệnh ở T+2.

Biến mục tiêu này có ý nghĩa trực tiếp: dương → MUA; âm → BÁN; gần 0 → NẮM GIỮ.

3.3. Mô hình 1: Hồi quy tuyến tính (Linear Regression – Baseline)

3.3.1. Cơ sở lý thuyết

Linear Regression giả định target_return là tổ hợp tuyến tính của 9 đặc trưng đầu vào:

ŷ = w₀ + w₁·z₁ + w₂·z₂ + ... + w₉·z₉

Trong hệ thống thực tế (ManualLinearRegression), các trọng số được tối ưu bằng Gradient Descent với learning_rate=0.01, iterations=1000:

∂L/∂w = (1/n) · Xᵀ · (ŷ − y)   →   w ← w − lr × ∂L/∂w
∂L/∂b = (1/n) · Σ(ŷ − y)       →   b ← b − lr × ∂L/∂b

L(w) = (1/n) · Σ (yᵢ − ŷᵢ)²   (MSE Loss)

3.3.2. Minh họa một bước Gradient Descent

Giả sử tại iteration đầu tiên, w = [0,0,...,0], b = 0 (khởi tạo):

ŷ₁ = 0 (dự báo ban đầu = 0)
y₁ = target_return = 0.00% (phiên 03/09: không đổi)
error₁ = ŷ₁ − y₁ = 0 − 0 = 0

ŷ₇ = 0, y₇ = +1.32%
error₇ = 0 − 0.0132 = −0.0132

∂L/∂w_close = (1/7) × Σ(errorᵢ × z_closeᵢ)
≈ (1/7) × [0×(−0.57) + ... + (−0.0132)×1.74]
≈ (1/7) × (−0.023) = −0.0033

→ w_close ← 0 − 0.01×(−0.0033) = +0.000033

Sau 1000 vòng lặp, các trọng số hội tụ. Giá trị trọng số đặc trưng sau huấn luyện:

w₀(bias) = +0.002    w₁(open) = +0.0041    w₂(high) = +0.0058
w₃(low) = +0.0029    w₄(close) = +0.0072    w₅(vol_norm) = +0.0031
w₆(rsi) = +0.0018    w₇(macd) = +0.0035    w₈(ma20) = +0.0012
w₉(volatility) = −0.0045   (biến động cao → kỳ vọng return giảm)

Nhận xét: w_close > w_high > w_open phản ánh đúng: giá đóng cửa và giá cao nhất là tín hiệu xu hướng mạnh nhất. w_volatility âm vì biến động cao thường đi kèm rủi ro, kéo giảm kỳ vọng lợi nhuận.

3.3.3. Dự báo cho phiên 12/09/2025

Sử dụng vector z-score của phiên 11/09 (hàng cuối tập Train) để dự báo return ngày 12/09:

ŷ = 0.002 + 0.0041×1.55 + 0.0058×1.85 + 0.0029×1.90
+ 0.0072×1.74 + 0.0031×1.74 + 0.0018×1.37
+ 0.0035×1.27 + 0.0012×0.38 + (−0.0045)×1.20
= 0.002 + 0.0064 + 0.0107 + 0.0055 + 0.0125 + 0.0054
+ 0.0025 + 0.0044 + 0.0005 − 0.0054
= +0.0465 ≈ +4.65%

Kết quả Linear Regression (dự báo T+2):

Đầu vào: Đặc trưng phiên T = 11/09/2025

Dự báo target_return(T+2): +4.65%

Thực tế target_return(T+2): +1.32% → (Close_15/09 − Close_11/09)/Close_11/09

Sai số MAE: |4.65% − 1.32%| = 3.33%

Mô hình Overestimate đà tăng vì không có bộ nhớ chuỗi thời gian và không nhận ra RSI = 72.4 (vùng quá mua) là tín hiệu cảnh báo.

Kết luận: Linear Regression hoạt động như một Baseline tham chiếu. Mô hình đơn giản, huấn luyện nhanh, nhưng không có bộ nhớ lịch sử và không bắt được phi tuyến tính của dữ liệu tài chính.
